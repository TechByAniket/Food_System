    About Our Project

    Project Name: Local Food Waste Reduction Platform

    This project aims to build a platform that connects food donors such as restaurants, hotels, and individuals with nearby NGOs or individuals in need. It allows users to list excess food, view listings in real-time, and manage requests for collection, reducing food waste and promoting community sharing.

    The platform features real-time food listings, role-based dashboards (Donor, NGO/Admin), and smart request management. It’s built using Python Flask for the backend, HTML/CSS with Tailwind for the frontend, and Supabase(PostgreSQL) for data management.

    We believe in using technology for social good — and this project reflects our passion for sustainability, responsibility, and smart community service.
